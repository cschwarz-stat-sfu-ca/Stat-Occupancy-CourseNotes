What is occupancy modelling?

The presence or absence of a species in a set of units (polygons, landscape units, territories, etc.) is a fundamental concept in many ecological studies (e.g., resource selection modelling, biodiversity, range). Visits to sampled units can result in a positive detection of a species or a non-detection of the species in that unit. However, a species may not always be detected if present which results in "false negatives". If the issue of detectablity is not accounted for, estimates that rely on the observed (naive) level of occupancy can be misleading.

Occupancy modelling is about methods for estimating species occurrence while accounting for potential false negatives.

A nice introduction is found at: USGS Occupancy Models to Study Wildlife

Software

There are several software packages that can be used for occupancy modelling.
Package Url
Presence
RPresence 		https://www.mbr-pwrc.usgs.gov/software/presence.html
Presence is a standalong Windoze program customized for the modelling of occupancy data.
RPresence is a R package that has the most common functionality of Presence and runs on Windoze, Mac, and Linus platforms.
GenPresencee https://www.mbr-pwrc.usgs.gov/software/presence.html
Used for planning occupancy studies.
When you install Presence, GenPresence is automatically installed in the same directory as Presence.
MARK
RMark 		http://www.phidot.org/software/mark/downloads/
MARK is a standalong Windoze program for capture-recapture, occupancy, and other studies.
RMark is an R packages available from CRAN that calls MARK and collects the output for further processing by R.
Because RMark calls MARK directly, it only runs under Windoze.

It is possible to install part of MARK on a Macintosh to run RMark natively without a virtualization engine.

unmarked An R package available from CRAN that has a limited set of occupancy models.
Because it is writting using S4, the documentation is extremely hard to follow.
JAGS Follow istructions at http://www.stat.sfu.ca/~cschwarz/CourseNotes/HowGetSoftware.html
A program to run Bayesian models written in BUGS. Commonly called from R to handle input/output.
R
Rstudio		 Follow istructions at http://www.stat.sfu.ca/~cschwarz/CourseNotes/HowGetSoftware.html
RPresence, RMark, unmarked, and JAGS use R extensively.

Courses

I offer a three day course on an Introduction of Occupancy Modelling covering the following topics. It is often offered through the Columbia Mountain Institute of Applied Ecology (http://www.cmiae.org). A sample offering is found at http://cmiae.org/event/occupancy-modelling-course/.

I can also do this or other courses onsite at your organization. Please contact me (Carl Schwarz) for information.

Slides from the course are available in this directory.

A typical course outline

  1. Introduction
    • differences/similarities of occupancy and capture-recapture modelling
    • fole of detectability
    • basic statistical review
    • concepts and notations
    • binomial data, odds ratios
    • basic likelihood
    • model selection and multi-model inference
  2. Single-season occupancy models
    • basic sampling protocol
    • PRESENCE/MARK/RPresence/RMark software to fitting single-season models
    • model assumptions
    • dealing with missing data
    • dealing with heterogeneity
    • more advanced features of PRESENCE/MARK/RPresence/RMark - design matrices/covariates
    • planning studies; GENPRES software
  3. Multiple-season occupancy models
    • sampling protocol
    • dynamics of population across seasons
    • using PRESENCE/MARK/RPresence/RMark
    • alternative parameterizations
    • characterizing occupancy dynamics
    • modelling spatial correlations in occupancy dynamics
    • dealing with missing data
    • dealing with covariates
    • study design
  4. Species co-occurrence models
    • sharing information among species
    • species richness or biodiversity
    • single season models/ multi season models
  5. Extensions
    • modelling spatial correlations in occupancy dynamics
    • incorporation of count data
    • incorporation of marked animals
    • incorporation of telemetry data
Additonal course covering more advance topics in occupancy modelling (e.g. bayesian methods, spatial methods, multi-method, multi-scale) can also be arranged. Please contact me for details.
